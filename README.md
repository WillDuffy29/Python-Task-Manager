# Python-Task-Manager
A command-line task manager built independently in Python. Users can add, remove, complete and view tasks. All data is saved between sessions using a JSON file. 

FEATURES

- Add tasks with a name, description, due date and priority.
- Remove tasks by name.
- Mark tasks as complete.
- View all tasks with their current status.
- Data persists between sessions via a JSON file.

HOW TO RUN

- Install Python onto your system.
- Clone or download the repository.
- Run main.py in your terminal.
- Follow the on-screen prompts.

CONCEPTS DEMONSTRATED

- Classes and objects
- Lists and loops
- Functions and modules.
- File handling (JSON).
- Data persistence.
- Control flow and user input.

PROJECT STRUCTURE

- tasks.py defines the 'Task' class.
- taskmanager.py defines the 'TaskManager' class.
- main.py handles the menu and user interaction.
- tasks.json is an automatically generated save file.
